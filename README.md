# Testing practice

> This project is about writting a few practical tests for JavaScript functions using the Jest library. 
## Built With

- JS
- Webpack

## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

- Code Editor (e.g. VS Code, Atom)
- Web browser 

### Setup

 **Step 1: Clone this repo locally using git**
- Create a folder in your local machine
- Open your console in your folder and run this command: `https://github.com/dqarias/testing-practice-jest.git`
- Alternatively, you can just download the complete zip file and extract the folder in your directory

 **Step 2: Run in your project directory**
- Open your console in your folder and run the nex command `. code` to open your vscode editor with awesomeBooks project.
- Then in index.html you can open with live server.
